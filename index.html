FoundationUltra is a product from relutionDev combining both foundationApps and foundationSites.

FoundationUltra is still very much in beta as there are some angular kinks to work out but untill then most elements seem to be working. If you resize the page you should see the topbar change with it whilst remaining functionional. Since ive seen alot of people ask about this on the forum it has been at the top of my todo list.

So Whats been Done

A bit of knowlege is required here so il give it to you. Foundation for sites uses jquery plugins and code to produce some of its responsive elements were as foundation for apps uses angular to modularise elements and animate/ control them. This means that without alot of refractoring foundationSites and foundationApps are incompatible with each other. But there was hope I had remembered reading about and angular port of foundationSites and so the fun began since then I have:

Creating a static version of foundationApps
Since foundationApps uses gulp to run on a test server during developement it specifies load_paths for the required angular and foundation apps scripts. To get around this I just declared the scripts into the header of the page including foundation.core and all of the foundation components
Removed All angular routing
Since the project I will be using this for will not contain angular for the MVC part of the application this part was not needed and so removed
Pulled in angular foundation aswell as foundationSites
